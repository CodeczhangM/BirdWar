# Level 1 - 遗忘地牢

## 概述

Level 1 是 Roguelike 地牢探索游戏的第一层，玩家将在程序生成的地牢中探索、战斗和收集宝藏。每次进入都会有不同的布局、敌人配置和战利品分布，提供高度的可重玩性。

## 文件结构

```
level1/
├── level1_config.json      # 主要关卡配置文件
├── level1_resources.json   # 资源清单和加载配置
├── level1_metadata.json    # 关卡元数据和分析信息
└── README.md              # 本文档
```

## 关卡特性

### 🎯 游戏目标
- **到达出口**: 找到通往下一层的楼梯
- **击败敌人**: 击败至少10个敌人
- **收集宝藏**: 收集3个宝箱（可选）
- **完美通关**: 不受伤害通过本层（可选）

### ⚔️ 敌人类���
- **骷髅战士** (40%): 基础近战敌人，血量50，伤害10
- **哥布林侦察兵** (35%): 快速敌人，血量30，伤害8，具有闪避能力
- **史莱姆** (20%): 坦克型敌人，血量40，伤害5，死亡时分裂
- **地牢老鼠** (5%): 群体敌人，血量20，伤害3，具有毒咬能力
- **骷髅队长** (精英): 强力敌人，血量150，伤害20，具有盾击和鼓舞能力

### 🎒 物品系统
- **武器**: 生锈的剑、铁剑、匕首
- **防具**: 皮甲、锁子甲、盾牌
- **消耗品**: 小型/中型生命药水、小型魔力药水
- **特殊物品**: 宝藏地图、地牢钥匙、传送卷轴

### 🎮 Roguelike 机制
- **程序生成**: 每次游戏都有不同的地牢布局
- **随机遭遇**: 敌人和宝藏位置随机生成
- **资源稀缺**: 有限的生命药水和装备
- **永久死亡**: 可选的硬核模式
- **元进度**: 解锁永久升级和奖励

## 房间类型

### 战斗房间 (50%)
- 3-6个敌人
- 30%概率包含宝箱
- 标准战利品掉落

### 宝藏房间 (15%)
- 0-2个敌人
- 100%包含宝箱
- 高价值战利品

### 商店房间 (10%)
- 无敌人
- 可购买物品和装备
- 价格随机波动

### 休息房间 (10%)
- 无敌人
- 恢复50点生命值
- 安全区域

### 精英房间 (10%)
- 1个精英敌人
- 80%概率包含宝箱
- 稀有战利品

### 陷阱房间 (5%)
- 2-4个敌人
- 包含隐藏陷阱
- 40%概率包含宝箱

## 程序生成系统

### 地牢生成
- **房间数量**: 8-15个房间
- **走廊长度**: 2-5格
- **房间大小**: 3x3 到 6x6
- **分支因子**: 30%

### 敌人生成
- **最小数量**: 15个敌人
- **最大数量**: 25个敌人
- **生成间隔**: 3秒
- **最大同时存在**: 8个敌人

### 战利品系统
- **金币掉落**: 根据敌人类型
- **物品掉落**: 基于掉落表概率
- **宝箱稀有度**: 普通、稀有、史诗

## Roguelike 设计原则

### 核心循环
1. **探索** - 发现新房间和路径
2. **战斗** - 击败敌人获取经验和战利品
3. **收集** - 获取装备和消耗品
4. **决策** - 选择前进路线和资源使用
5. **进步** - 到达下一层或死亡重来

### 风险与回报
- 探索更多房间 = 更多战利品 + 更多危险
- 挑战精英敌人 = 稀有装备 + 高风险
- 使用消耗品 = 即时收益 + 资源消耗

### 元进度系统
- 永久解锁新物品
- 角色升级保留
- 起始奖励增加
- 成就奖励累积
